# Fusão UX/Pedagógica: STUDEI 2.0 + Sistemão
**Data:** 2026-03-18 | **Status:** Planejamento

## Objetivo

Manter o motor FSRS (superior) e adicionar a **camada de orquestração pedagógica** que tornava o STUDEI 2.0 claro e eficaz. O usuário precisa saber o que fazer, ter contexto antes de estudar, e sentir que o sistema o está guiando.

## O que muda (visão geral)

```
ANTES:  Dashboard → [Botão "Iniciar"] → /study → cards FSRS
DEPOIS: Dashboard → Missão Diária com HERO → [Estudar Tópico X] 
                  → Tela Pré-Voo (Active Recall + Bizus + Erros) 
                  → Cards FSRS do tópico → Resultado do tópico
```

### Proposed Changes

#### [NEW] PreVoo.svelte — `src/lib/components/study/PreVoo.svelte`
- Componente de tela completa
- Mostra: matéria, tópico, tipo de sessão (Novo / Revisão)
- Se **Revisão**: exibe Active Recall

- Se **Novo**: exibe ritual simples (checkbox celular, foco, água)
- Botão "Pular" disponível (sem XP)
